# ArtiFlux Privacy Policy

The privacy policy page for the ArtiFlux application.

## Deployment

This site is deployed to GitHub Pages using GitHub Actions.

To enable deployment:

1. Go to the **Settings** tab of this repository.
2. Select **Pages** from the sidebar.
3. Under **Build and deployment** > **Source**, select **GitHub Actions**.
4. The deployment will happen automatically on the next push to `main`.
